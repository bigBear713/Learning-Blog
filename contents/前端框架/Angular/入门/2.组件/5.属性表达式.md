## 效果
- 在模板中动态加载组件ts中的变量值。当组件中的变量值改变时，模板中的数据也能跟着改变。

```ts
@Component({ template: `<p [title]="str" [style.color]="color">text</p>` })
export class XXXComponent {
  str = 'hello world';
  public color='red';
}
```
- 显示效果
```html
<p title="hello world" style="color: red;">text</p>
```

## 语法
- 使用`[]`将需要动态数据的属性包裹起来。
- 引用的组件中的变量需要为`public`。
